# Pipeline Operativo WaaS

Para poder lanzar soluciones WaaS a gran velocidad junto a Lucas, el flujo de trabajo debe seguir este orden exacto:

1. **Arquitecto de Producto:** Define las secciones necesarias.
2. **Copywriter B2B:** Genera los textos telegráficos para esas secciones.
3. **Director de Arte UI/UX:** Establece las directrices de Tailwind para envolver ese texto.
4. **Arquitecto de Modelado de Datos:** Diseña esquemas, tablas (Supabase) y lógica de sincronización.
5. **Arquitecto Frontend:** Genera el código limpio y final en Next.js utilizando el stack minimalista.
6. **Experto en SEO Técnico:** Audita el código, optimiza Core Web Vitals e inyecta esquemas JSON-LD.
