# 🎨 Taste: Principios Editoriales y Minimalismo Cálido

Este agente garantiza que LUNAKIDS mantiene una estética de alta fidelidad, alejándose de los patrones "por defecto" (slop) de la IA.

## 1. Minimalismo Cálido (Estética LUNAKIDS)
- **Paleta Intocable:** Usa exclusivamente los colores de la marca (lino, marengo, cerceta/verde salvia, granate, terracota).
- **Prohibido el "AI Purple" o azules eléctricos:** Nunca uses gradientes violetas ni estilos "cyber/SaaS".
- **Contraste Orgánico:** Nunca uses negro puro (`#000000`) ni blanco puro (`#FFFFFF`). Utiliza fondos crema o lino, y textos gris marengo u ónix.

## 2. Tipografía y Jerarquía
- **Serif vs Sans:** Mezcla `font-serif` (Cormorant) para titulares elegantes y `font-sans` (Montserrat/Inter) para legibilidad en cuerpos de texto.
- **Micro-Tipografía (Eyebrows):** Usa `uppercase tracking-[0.2em] text-[11px]` para etiquetas, pero no abuses (máximo 1 o 2 por página).
- **Altura de Línea:** Para grandes titulares, usa `leading-none` o `leading-tight`. Para párrafos legibles, `leading-relaxed`.

## 3. Uso del Espacio (White Space)
- **Grid Asimétrico:** Evita diseños perfectamente simétricos de 3 columnas iguales si puedes usar grids 2/1 o asimetría "Bento" (Grid CSS).
- **Respiración:** Usa `py-24` o `py-32` para separar secciones clave. El contenido nunca debe sentirse asfixiado.
- **Card Containers:** Evita meterlo todo en cajas blancas con sombras genéricas. Si el diseño lo permite, deja que la tipografía y las imágenes respiren sobre el fondo principal.
